Title: Rotten Pumpkin Website

Medium: html5, css

Year: 2019

In Art74 we learned the basics towards building websites! As a 3rd grader in elementary school before the start of social media, a friend introduced me to a website builder. I remember obsessively looking for ways to decorate my page using codes designated for myspace and embeding flash games to share with my friends. Using elements such as custom cursors and decorative sprite gifs inspired the aesthetic behind the website. Halloween was an upcoming holiday around the production of this website which is why it is Halloween themed, and creating a parody of the website Rotten Tomatoes inspired the purpose for Rotten Pumpkin! Users would rate their favorite Halloween-themed movies using the rating widget and are encouraged leave reviews.
